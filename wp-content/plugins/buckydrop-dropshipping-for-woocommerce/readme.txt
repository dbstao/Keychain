== Installation ==

= From within WordPress =

1. Visit 'Plugins > Add New'
2. Search for 'BuckyDrop'
3. Activate BuckyDrop from your Plugins page.
4. Go to "after activation" below.

= Manually =

1. Upload the `BuckyDrop` folder to the `/wp-content/plugins/` directory
2. Activate the BuckyDrop plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
